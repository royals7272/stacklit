## 🛠️ What stacklit does

AI agents like Claude Code, Cursor, and Aider need help to understand large sets of files. Normally, they read too much data. This takes time and costs money. Stacklit acts like a librarian. It scans your folders and creates a clear summary of your project structure. 

When you use stacklit, an AI agent only looks at the relevant pieces of your project. It ignores the clutter. It focuses on the code that matters. This tool works well with large projects that contain many folders and files. It connects directly with your existing AI setup to make the workflow smooth.

## 📁 Managing your codebase map

Your project files change often. Stacklit tracks these changes. If you add new files or update existing code, run stacklit again. It updates the index file. The AI agent notices the change and stays current with your latest work.

You can run this process as often as you want. There is no limit to how many projects you can index. The tool keeps your workspace clean by organizing the data in the background. It does not change your code. It only reads the files to build the map.

## 🔍 Troubleshooting common issues

If the AI fails to see the project files, check these items:

1. Refresh the index. Sometimes the AI needs a moment to see the new data. Click the refresh button in your AI tool.
2. Check your file path. Make sure you selected the correct folder in the stacklit tool.
3. Ensure the AI tool has permission to access your files. 
4. Close and restart your AI interface. This forces a fresh look at your project files.

Stacklit runs as a background process. It uses very little power. You can leave it open while you work on other tasks. It waits until you trigger it to index your files. 

## 🛡️ Privacy and data

Your code stays on your computer. Stacklit generates the index file locally. It does not send your private code to remote servers or third-party databases. Only the AI tool you authorize uses the map that stacklit creates. This ensures your project remains private while you use external AI agents for assistance.

If you ever wish to remove the tool, go to your Windows Settings, select Apps, and find stacklit in the list. Choose uninstall to remove the program and its settings from your computer. Your code files remain untouched during this process.
